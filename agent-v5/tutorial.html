Tutoriel

Comment utiliser l’agent

1. Remplir Mes données avec identité, compétences, expériences, éventuellement URL LinkedIn et fichier source.
2. Remplir le Mode recruteur avec le poste, le secteur, l’entreprise, le salaire et le texte d’offre ou les notes associées.
3. Consulter Matching pour observer le score local, le recoupement et la synthèse.
4. Utiliser le Studio pour générer plusieurs versions selon le type de document et le rendu souhaité.
5. Vérifier le rendu dans Aperçu avant toute réutilisation ou export.
